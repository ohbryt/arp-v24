# 초록 (Abstract)

에스트로겐 관련 수용체 감마 (Estrogen-Related Receptor Gamma, **ERRγ**, 유전자명 **ESRRG**, 핵수용체 번호 **NR3B9**)는 orphan 핵수용체에 속하며,迄今为止는 명확한 내인성 리간드가 보고되지 않은 핵수용체 계열이다.

ERRγ는 **PGC-1α**와의 상호작용을 통해 미토콘드리아 생합성, 산화적 인산화, 지방산 산화, 그리고 갈색지방 발열작용을 포함한 대사 조절에 핵심적인 역할을 한다.

본 보고서는 ERRγ의 구조 및 동형체 isoform 특성, 조직 분포, 대사 조절 기전, 질환 맥락에서의 역할, 그리고 현재까지 보고된 합성 작용제들을 종합적으로 서술한다.

# 1. 서론: ERRγ 생물학 (Introduction: ERRγ Biology)

## 1.1 ERRγ 발견 및 명칭 기원

에스트로겐 관련 수용체 (Estrogen-Related Receptors, ERRs)는 에스트로겐 수용체 (Estrogen Receptors, ERs)와 구조적 유사성을 가지지만, 내인성 에스트로겐에는 결합하지 않는 핵수용체 초가족의 구성원이다.

| Isoform | 유전자명 | 핵수용체 번호 | 발견 연도 |
|---------|----------|--------------|----------|
| ERRα | NR3B1 | 1988 |
| ERRβ | NR3B2 | 1988 |
| **ERRγ** | **NR3B9** | **1999** |

> **"Orphan receptor"** = 내인성 리간드 미확인 수용체

ERR family는 자연적으로 리간드에 의존하지 않고 **구성적 활성화 (constitutively active)** 될 수 있으며, 이는 AF-2 도메인이 공동활성자의 결합을 가능하게 하는 활성 상태를 본질적으로 채택하고 있음을 의미한다.

## 1.2 ERRγ 구조 및 Isoform 특성

### 1.2.1 핵수용체 도메인 구조

```
┌─────────────────────────────────────────────────────────────┐
│                    ERRγ Domain Structure                     │
├─────────────────────────────────────────────────────────────┤
│  A/B │    DBD    │  Hinge  │         LBD (with AF-2)    │
│  NTD │  (Zinc×2) │          │    Helix 12 = AF-2 핵심    │
└─────────────────────────────────────────────────────────────┘
```

| 도메인 | 기능 |
|--------|------|
| **A/B (NTD)** | 조직-특이적 발현, 공동활성자 상호작용 |
| **DBD** | ERR Response Element (ERRE) 결합 |
| **Hinge** | DBD-LBD 연결 |
| **LBD** | 리간드 결합, AF-2 도메인 포함 |

### 1.2.2 세 가지 ERR Isoform 비교

| 특성 | ERRα | ERRβ | **ERRγ** |
|------|-------|-------|-----------|
| 유전자 위치 | 11q13 | 1q42 | 1q42 |
| LBP 크기 | ~100 Å³ | 중간 | ~200-300 Å³ |
| **주요 발현 조직** | 갈색지방, 장, 골격근 | 눈, 귀, 신장 | **심장, 뇌, 골격근, 췌장** |
| 내인성 리간드 | 미동정 (orphan) | 미동정 | **미동정** |
| PGC-1α 공동활성 | ○ | ○ | **○** |
| 구성적 활성화 | 높음 | 중간 | **중간-높음** |

## 1.3 조직 분포 (Tissue Distribution)

ERRγ는 전형적으로 **고에너지 요구 조직**에서 높게 발현된다:

| 조직 | 발현 수준 | 주요 기능 |
|------|----------|----------|
| **심장** | 가장 높음 | 미토콘드리아 산화적 인산화, 지방산 산화 |
| **뇌** | 높음 | 에너지 항상성, 신경전달물질 대사 |
| **골격근** | 높음 | Oxidative fiber 규제, 지구성 운동 적응 |
| **갈색지방** | 중간-높음 | 비哆嗦성 발열 (UCP1) |
| **췌장** | 중간 | β세포 인슐린 분비 |
| **신장** | 중간 | 에너지 항상성 유지 |

## 1.4 DNA 결합 (DNA Binding)

### 1.4.1 ERR Response Element (ERRE)

```
 Consensus: 5'-TNAAGGTCA-3' (합성 호몰로듀플렉스)
```

ERRγ가 결합하는 대표적 유전자:
- **PGC-1α** 프로모터
- **PDK4** 엔한서
- **MCAD** (지방산 산화 관련)
- **ATP5b** (ATP Synthase)

## 1.5 공동활성자 모집 (Coregulator Recruitment)

ERRγ의 핵심 특성: **리간드 비의존적 활성화**

핵심 공동활성자:
- **PGC-1α** (가장 중요) → 미토콘드리아 생합성
- **SRC-1, SRC-2, SRC-3** ( Steroid Receptor Coactivators)

# 2. ERRγ의 대사 조절 역할 (ERRγ in Metabolic Regulation)

## 2.1 미토콘드리아 생합성 (Mitochondrial Biogenesis)

### 2.1.1 PGC-1α协同 활성화

```
에너지 요구/운동 → AMPK 활성화 → PGC-1α 인산화 → ERRγ 활성화
                                                            ↓
                                    미토콘드리아 생합성 유전자 전사 활성화
                                                            ↓
                                          미토콘드리아 품질 및 수량 증가
```

### 2.1.2 미토콘드리아 전사 인자들

| 단백질 | ERRγ/PGC-1α 조절 | 기능 |
|--------|-----------------|------|
| **ATP synthase** | 활성화 | 산화적 인산화 |
| **cytochrome c** | 활성화 | 전자전달계 |
| **COXI-IV** | 활성화 | 복합체 IV |
| **ND1, ND2, ND4** | 활성화 | 복합체 I |
| **UCP1** | 조직-의존적 | 열발생/해리 |

## 2.2 산화적 인산화 (OXPHOS)

ERRγ 전임상 연구 결과:

| 측정 지표 | 효과 |
|----------|------|
| 산화적 인산화 용량 | **↑ 증가** |
| 산소 소비율 (OCR) | **↑ 상승** |
| ATP 생산 효율성 | **↑ 개선** |
| 복합체 I, III, IV, V subunits | **↑ 발현 증가** |

## 2.3 지방산 산화 (FAO)

ERRγ 활성화 시 증가하는 효소:

| 효소 | 기능 |
|------|------|
| **CPT1** | 미토콘드리아로의 지방산 유입 조절 |
| **MCAD** | β-산화의 첫 단계 |
| **PDK4** | 포도당 산화 억제, 지방산 산화 촉진 |

> **SLU-PP-332 투여 시 PDK4 전사: 2-4배 증가** (Billon et al., 2023)

## 2.4 갈색/베이지 지방 발열 (Thermogenesis)

```
Cold exposure → Norepinephrine → ERRγ 발현 증가
                                    ↓
         UCP1 발현 ↑ + 미토콘드리아 생합성 ↑ + 발열 유전자 프로그램 활성화
                                    ↓
                          체온 유지 및 에너지 소비
```

# 5. ERRγ 활성화의 구조적 기반 (Structural Basis)

## 5.1 AF-2 도메인 구조

```
              Helix 12 (AF-2 핵심)
                   ↓
    ┌─────────────────────────────┐
    │      ERRγ LBD Structure       │
    │                              │
    │  H1-H3    H4-H5    H6-H7   │
    │    ↓        ↓        ↓      │
    │  ┌────────────────────────┐  │
    │  │   Ligand-Binding      │  │
    │  │      Pocket          │  │
    │  │   (200-300 Å³)       │  │
    │  └────────────────────────┘  │
    │                    H8  H9-H10│
    └─────────────────────────────┘
```

핵심 잔류:
- **Leu345**: hydrophobic pocket 구성
- **Phe377**: aromatic interaction
- **Glu384**: 리간드의 hydroxyl group과 H-bond
- **Tyr326**: aromatic, π-π interactions

## 5.2 리간드 비의존적 활성화

```
PGC-1α 결합 → ERRγ AF-2 표면 안정화 → H12 활성 위치 
                                                    ↓
                              전사 공진 복합체 형성 → 미토콘드리아 생합성
```

> ERRγ는 리간드 없이도 **PGC-1α에 의해 활성될 수 있음** = 구성적 활성화

# 10. 결론 (Conclusion)

ERRγ (ESRRG, NR3B9)는 orphan 핵수용체로 분류되지만, **PGC-1α와의 밀접한 상호작용**을 통해:

- ✅ 미토콘드리아 생합성 촉진
- ✅ 산화적 인산화 능력 향상
- ✅ 지방산 산화 촉진
- ✅ 에너지 항상성 조절

**SLU-PP-332**는 현재까지 가장 광범위하게 연구된 pan-ERR 작용제로:

| 효과 | 증거 수준 |
|------|----------|
| 운동 모방 효과 | 전임상 (강력) |
| 대사 개선 | 전임상 (강력) |
| 심장 보호 | 전임상 (中等) |
| 노화 방지 | 전임상 (初步) |
