## Executive Summary

당뇨병성 신병증(Diabetic Nephropathy, DN)은 당뇨병의 주요 합병증으로, 세계적으로 **말기신장병(ESRD)의 주요 원인**이다. 현재 치료제는 **SGLT2 억제제**와 **GLP-1 작용제**가 있으나, **섬유화를 직접 억제하는 약물은 부재**한状况이다.

**LOXL2**는 다음과 같은 이유로 **DN의 새로운 치료 표적**으로 각광받고 있다:
- 사구체 및 세뇨관간질 섬유화 억제
- TGF-β/LOX/Snail 축 차단
- 상피-간葉 중간형 전환(EMT) 억제
- 현재 신약 개발 기회가 충분 (Simtuzumab은抗体形式로失敗)

## 1. 당뇨병성 신병증 (DN) 개요

### 1.1 질병 개요

| 항목 | 내용 |
|------|------|
| **유형** | 1형 및 2형 당뇨병 합병증 |
| **유병률** | 당뇨병 환자의 30-40% |
| **진행** | 알부민뇨 → 사구체섬유화 → ESRD |
| **시장규모** | ~$50B (글로벌) |
| **批准治療** | SGLT2 억제제, GLP-1 작용제 |

### 1.2 질병 단계

| 단계 | 특징 | 임상 지표 |
|------|------|----------|
| ** Stage 1** | 고혈압/과혈당 초기 | 정상 알부민 |
| ** Stage 2** | 초기 신장병변 | 미세알부민뇨 (30-300 mg/day) |
| ** Stage 3** | 임상적 알부민뇨 | 거대알부민뇨 (>300 mg/day) |
| ** Stage 4** | 진행성 섬유화 | 사구체/세뇨관 섬유화 |
| ** Stage 5** | 말기신장병 (ESRD) | 투석/이식 필요 |

### 1.3 현재 치료 현황

| 약물 | 표적 | 효과 | 한계 |
|------|------|------|------|
| **SGLT2 억제제** | 신장에서 당 회수 억제 | 알부민뇨 감소, 신기능 보호 | 섬유화 직접 억제X |
| **GLP-1 작용제** | GLP-1 수용체 | 혈당/체중 감소 | 섬유화 직접 억제X |
| **RAS 차단제** | Angiotensin II | 혈압/알부민뇨 | 완전한 섬유화 억제 어려움 |
| **Fibrate** | PPAR-α | 지질 개선 | 신보호 효과 제한적 |

## 2. LOXL2와 신장 섬유화 기전

### 2.1 LOXL2의 생물학적 기능

**LOXL2 (Lysyl Oxidase-like 2):**
- **분류:** Amine oxidase enzyme
- **주요 기능:**
  1. **교차결합 촉진:** Collagen과 elastin의 lysine 잔기를 산화하여 섬유 간 교차결합 형성
  2. **EMT 촉진:** 상피세포를 중간형으로 전환시켜 섬유모세포 활성화
  3. **전사 조절:** "Moonlighting" 기능으로 핵 내 전사 인자로서 작용

### 2.2 TGF-β/LOX/Snail 축 (2025 Paper)

```
고혈당 (Hyperglycemia)
      ↓
TGF-β 활성화
      ↓
LOX/LOXL2 발현 ↑
      ↓
Snail 전사인자 활성화
      ↓
상피-간葉 중간형 전환 (EMT)
      ↓
세뇨관세포 손상 + 섬유모세포 활성화
      ↓
세뇨관간질 섬유화 (Tubulointerstitial Fibrosis)
```

**Paper (2025):** "LOX-induced tubulointerstitial fibrosis via the TGF-β/LOX/Snail axis in diabetic mice" (J Transl Med, Jan 2025)

### 2.3 사구체에서의 역할

| 세포 유형 | LOXL2 효과 |
|-----------|-----------|
| **사구체세포** | 사구체基底막 합성 증가 |
| **세뇨관세포** | EMT 유도, 섬유화 촉진 |
| **미세혈관세포** | 내피 dysfunction |
| **침윤 면역세포** | 염증 반응 강화 |

### 2.4 신장 질환에서의 증거

| 연구 | 모델 | 결과 |
|------|------|------|
| **2018 (Nature)** | 당뇨쥐 | LOXL2 억제가 사구체섬유화, 알부민뇨 개선 |
| **2025 (J Transl Med)** | 당뇨 mice | TGF-β/LOX/Snail 축 차단이 섬유화 역전 |
| **2025 (Springer)** | TGF-β/LOX | LOX 특이적 억제가 신장 보호 효과 |
| **2021 (PMC)** | Human | LOXL2 발현이 IgA 신병증에서 증가 |
| **2018** | Rat | LOXL2 발현이 당뇨병성 신병증에서 ↑ |

## 3. LOXL2 억제제 개발 현황

### 3.1 저분자 억제제

| 화합물 | 개발사 | 단계 | 특징 |
|--------|--------|------|------|
| **SNT-5382** | - | Preclinical | LOXL2 선택적 억제제 |
| **PXS-5120A** | Pharmaxis | Preclinical | Pan-LOX 억제제 (LOX + LOXL2) |
| **BAPN** | - | Research | 최초 LOX 억제제 (BAPN: β-aminopropionitrile) |

### 3.2 항체 억제제

| 항체 | 개발사 | 단계 | 결과 |
|------|--------|------|------|
| **Simtuzumab (GS-6624)** | Gilead | Phase 2 | **실패** (간섬유증, 췌장암) |
| **AB0023** | - | Preclinical | 동물모델에서 효과 확인 |

### 3.3 Simtuzumab 실패의 교훈

**Phase 2 실패 이유:**
1. **항체 형식의 한계:**Large molecule, 조직 침투 제한
2. **적응증 선택 오류:** 간섬유증에서 주로 시험
3. **환자 선택:** 중증도가 높은 환자 대상
4. ** dosing 문제:** 노출량 부족 가능성

**Small molecule 우위:**
- 조직 침투 우위
- 낮은 비용으로 높은 doses 가능
- 신장 특이적 치료 가능

## 4.Combination Therapy 전략

### 4.1 SGLT2 억제제 + LOXL2 억제제

| 약물 | 표적 | 기전 |
|------|------|------|
| **SGLT2 억제제** | SGLT2 | 혈당↓, 알부민뇨↓, 사구체 과여과↓ |
| **LOXL2 억제제** | LOXL2 | 섬유화 직접 억제, EMT 차단 |

**병용 이점:**
- SGLT2: 대사 개선 + 초기 신장 보호
- LOXL2: 진행된 섬유화 억제
- **상승효과 (Synergistic):** 대사 + 구조적 보호

### 4.2 GLP-1 + LOXL2 억제제

| 약물 | 표적 | 기전 |
|------|------|------|
| **GLP-1 작용제** | GLP-1R | 혈당↓, 체중↓, 염증↓ |
| **LOXL2 억제제** | LOXL2 | 섬유화 직접 억제 |

### 4.3berberine + LOXL2 억제제

**berberine의 효과:**
- AMPK 활성화
- TGF-β/Smad 경로 억제
- 혈당 개선
- **LOXL2 억제제와 병용 시 시너지 가능**

## 7. 임상 개발 로드맵

### Phase 1 (2026-2027)
- **Preclinical:** LOXL2 억제제 in vitro/vivo validation
- ** Toxicology:** GLP toxicology studies
- **Formulation:** 신장 표적 전달 시스템 개발

### Phase 2a (2027-2028)
- **2a 연구:** 2형 당뇨병성 신병증 환자 50-100명
- **终点:** 알부민뇨 감소, 신기능 지표
- **병용:** SGLT2 억제제와 병용

### Phase 2b (2028-2029)
- **2b 연구:** 200-500명, 다기관
- **终点:** 사구체여과율 (eGFR) 감소 속도
- **병용:** Standard of care (SGLT2 + RAS 차단)

### Phase 3 (2030-2032)
- **3상:** 1000+ 명, 글로벌
- **终点:** ESRD 진행, 투석 시작
- **예상:** 30-40% 섬유화 진행 억제
