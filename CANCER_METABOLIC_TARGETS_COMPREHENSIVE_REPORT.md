## 3.3 L-Asparaginase Therapy

| 약물 | 적응증 | 반응률 | 상태 |
|------|--------|-------|------|
| **L-아스파라기나제 (E. coli)** | ALL | ~80-90% | Approved (1978) |
| **PEG-아스파라기나제 (Oncaspar)** | ALL | ~90% | Approved (1994) |
| **Erwinase** | ALL (불내성) | ~80% | Approved (2011) |

```
L-아스파라기나제 내성 = ASNS upregulation (60-70%에서 발생)

```

## 3.5 Key Finding

```
ALL 치료의 표준 요법 L-아스파라기나제의 내성 극복에 Metformin 조합 권장

```

```
Ketogenic diet → Ketone bodies ↑ → GPR81 활성화 → 대사 개선
Ketone bodies: β-Hydroxybutyrate (2-5 mM), Acetoacetate (0.5-2 mM)
```

## 5.5 Key Finding

```
GPR81은 두 가지 역할:
- 암: GPR81 저해제로 면역회피 차단 가능성
- 대사: Ketogenic diet로 대사 개선 가능
```
